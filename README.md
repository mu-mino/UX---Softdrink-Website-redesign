# FuseTea – UX/UI Case Study

Ein vollständiges UX/UI-Redesign für einen Softdrinks-Onlineshop, entstanden im Rahmen eines einwöchigen Design Sprints.

## Inhalt

| Seite | Beschreibung |
|-------|-------------|
| `index.html` | Hub-Seite: Projektziel, Before/After, Rollen, Kapitelübersicht |
| `pages/backstory.html` | Cartoon-Reihe: Wie das Projekt entstand |
| `pages/ia.html` | Arbeitsprozess I – Informationsarchitektur & Wireframes |
| `pages/produkte.html` | Arbeitsprozess II – Produktbilder-Redesign |
| `pages/empathie.html` | Arbeitsprozess III – Empathy Maps, Personas, HMW |
| `pages/texte.html` | Arbeitsprozess IV – Textoptimierung & Zielgruppenansprache |
| `pages/rueckblick.html` | Sprint Review, Post-Launch Feedback & Learnings |

## GitHub Pages

URL: "https://mu-mino.github.io/UX---Softdrink-Website-redesign/" 

## Stack

- Vanilla HTML/CSS/JS – kein Framework, kein Build-Tool
- [anime.js](https://animejs.com/) – Seitenübergangs-Animation
- [TwentyTwenty](https://github.com/zurb/twentytwenty) – Before/After Slider
- Lokale Schriften: Riffic (Cartoon), Montserrat, Fredoka (jeweils Variable Fonts)

## Projektkontext

**Rolle:** UX Design, UI Design, Visual Design, Grafik Design, UX Writing, Informationsarchitektur, Responsive Design, Projektmanagement  
**Zeitrahmen:** 1 Woche (Design Sprint)  
**Tools:** Krita, Canva, Goodnotes, ChatGPT, Galerie-App (Samsung)
